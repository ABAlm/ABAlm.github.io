My Projects

Super RoboCar

The Super Robocar is an autonomous robot that navigates its surroundings while providing real-time visual and audio feedback. It features an HC-SR04 ultrasonic sensor for obstacle detection, a 16x2 I2C LCD display for visual distance output, and a DFPlayer Mini module for engine-like audio feedback. Each team member customized the robot to resemble their favorite super sports car. The project integrates these three key modules to ensure smooth functionality. Future improvements include transforming the Super Robocar into a Bluetooth-controlled device for enhanced remote operation.
